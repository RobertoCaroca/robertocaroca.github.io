# robertocaroca.io

Project Title: Roberto Caroca Portafolio
https://robertocaroca.github.io/

Description of Project: Este projecto es sobre el desarrollo de mi sitio web para mostrar los proyectos en los que he trabajado para obtener el Certificado Profesional en Programación: Desarrollo Full-Stack con MERN de MIT.

Version: 1.0
Version 1.0 description: Esta es la primera version de mi portafolio la cual incluye los siguientes trabajos:
   1. Pacman Exercise: Ejercicio para construir una fabrica de PacMan
   2. Eye Exercise: Ejercicio para interactuar con el curso de tu pantalla a traves de unos ojos que no dejan de seguirlo.
   3. Real Time Bus Tracker Exercise: Ejercicio para obtener data externa que muestre las parada de una autobus en tiempo real.
